# Keikai
Swift framework for displaying message in app.


## Table of Content

 - [Requirements](#requirements)
 - [Contribution](#contribution)
 - [License](#license)


### Requirements

- Xcode 10+
- Swift 5+
- iOS 10+

### Usage

Show a simple message

```

	let keiKaiView = KeikaiView(message: "Hello World!", duration: .short)
    keiKaiView.show()
	
```
